# I Smoke Window
'I Smoke Window' is a simple react app made to help you learn utilities on CS2. 

# Dev 
Quick guide on how to launch the project localy 
## Prerequisites
 * node : v22.14.0
 * npm : v10.9.2

## Launching localy 
Git Clone the project on your machine. 
Go to the 'i-smoke-window' folder. 
Run 'npm install'
Run 'npm -start'
